# Content Repurposer

Take one piece of content and repurpose it across multiple platforms with platform-optimized formats.

## Instructions

1. Read `.claude/skills/blotato-social-media/SKILL.md` for platform rules
2. Read `brand.md` for brand voice
3. Extract or analyse the source content:
   - YouTube/video: extract transcript via Blotato, summarise key insights
   - Blog/article: identify 3-5 key takeaways
   - Podcast: extract transcript, pull quotes and insights
4. Repurpose into platform-specific formats:
   - LinkedIn: 600-word professional post
   - Instagram: carousel slide texts (3-5 slides)
   - X/Twitter: 3 standalone tweets or thread
   - TikTok: short-form video script
   - Pinterest: keyword-rich pin description
5. Schedule repurposed content across the week (not all on same day)
6. Report: what was created, where it's scheduled, suggested visuals
