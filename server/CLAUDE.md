## Server (Backend)

- Express + tRPC + TypeScript
- MySQL + Drizzle ORM
- LINE Messaging API webhook at /webhook
- Google Gemini LLM via OpenAI-compatible endpoint
- 8891.com.tw vehicle inventory sync
- Vehicle detection service with Map-based index
- 2-min TTL cache on getAllVehicles()
- AES-256-GCM PII encryption
